# 📱 Mobile Sales Analytics Dashboard

## Overview
This Power BI project provides an interactive analysis of mobile sales data, helping businesses track sales performance, customer behavior, and product trends through dynamic visualizations.

## Key Features
- Sales Performance Analysis (₹769M+ Total Sales)
- Product-wise and Brand-wise Analysis
- Monthly Sales Trends
- Customer Rating Insights
- Payment Method Distribution
- City-wise Sales Analysis
- Interactive Filters and Slicers

## Tools Used
- Power BI
- DAX
- Excel / CSV
- Data Modeling

## Dashboard Pages

### Page 1: Sales Overview
- KPI Cards
- Monthly Sales Trends
- Sales by City
- Customer Ratings Analysis
- Payment Method Distribution

### Page 2: Product Analysis
- Mobile Model Performance
- Brand Distribution
- Customer Rating vs Sales Analysis
- Product-wise Transaction Analysis

## Key Insights
- Analyzed over ₹769M in sales transactions.
- Identified top-performing mobile brands and models.
- Evaluated customer ratings and purchasing patterns.
- Tracked monthly sales trends and payment preferences.
